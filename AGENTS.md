# AGENTS.md

## 项目定位

本仓库是一个面向中国互联网 AI 数据产品经理岗位的作品集，用于展示：

1. 智能问数
2. 自然语言转 SQL
3. 自动归因分析
4. Prompt 调优
5. AI workflow 原型设计
6. 评测体系设计

本仓库不是算法竞赛项目，也不是复杂工程项目。优先展示产品理解、数据分析流程、AI 能力边界和评测思维。

## 工作原则

1. 所有代码必须简单、可运行、适合 Python 初学者理解。
2. 不要过度工程化。
3. 不要引入复杂框架，除非明确要求。
4. 默认使用 Python、pandas、Streamlit。
5. 所有 README 和文档使用中文。
6. 文档表达贴近中国互联网数据产品面试语境。
7. 不使用 emoji。
8. 代码中不要出现装饰性图标。
9. 每个项目必须包含 README、运行方式、功能说明、测试方式和面试讲法。
10. 每次修改后，必须说明修改了哪些文件，以及如何运行验证。

## TDD 工作原则

1. 新增功能前，先写单元测试。
2. 每个测试必须明确输入、预期输出和业务含义。
3. 实现代码只做让测试通过所需的最小功能。
4. 每轮修改后必须执行单元测试，默认使用 `pytest`。
5. 如果本轮没有业务代码，也必须执行结构检查或说明为什么不运行单元测试。
6. 最终说明必须包含测试命令和测试结果。
7. 不为了通过测试而写脱离业务语境的硬编码逻辑。
